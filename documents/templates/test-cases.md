# Test Cases

> Functional QA suite for one module. Master table below = the `Test_Cases` sheet.
> Generate with `skills/testing/qa-test-cases.yaml`; export to the 5-sheet workbook
> (`00 Overview · PCL · Test_Cases · Test_Run · Bugs`) via `tools/qa/build_workbook.py`.

- **IDs:** PCL `{APP}-{MODULE}-PCL-NNN` · TC `{APP}-{MODULE}-TC-NNN`
- **Type:** Functional / Negative / Boundary / Integration / Regression (+ project extras)
- **Priority:** Critical / High / Medium / Low
- **Rules:** every Expected can FAIL · steps UI-executable · concrete data · no silent
  assumptions (tag `[ASSUMPTION]` with reason, or `[Pending]`).
